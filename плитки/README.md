# плитки

Batch-скрипт для Windows, который:

- блокирует перемещение плиток меню «Пуск»
- либо возвращает возможность перемещения плиток

Скрипт меняет параметры реестра и перезапускает `explorer.exe`, чтобы применить изменения.

## Запуск

1. Открой `плитки/плитки.bat`
2. Запусти **от имени администратора**
3. Выбери нужный режим в меню

## Примечания

- Работает через изменения реестра, поэтому требуются права администратора.
- Если используешь на другой версии Windows и поведение отличается — сначала сделай точку восстановления/бэкап реестра.
