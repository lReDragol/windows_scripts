# очистка_комментариев_py

GUI-утилита (Tkinter) для обработки `.py` файлов:

- удаляет `#` комментарии (кроме shebang и coding cookie)
- опционально удаляет docstring-и (module/class/function)
- опционально удаляет `/*...*/` внутри QSS тройных строк (если переменная содержит `QSS`)
- (опционально) делает резервную копию рядом с файлом (`.bak_YYYYmmdd_HHMMSS`)

## Запуск

```bash
python очистка_комментариев_py/очистка_комментариев_py.py
```

## Как пользоваться

1. После запуска выбери один или несколько `.py` файлов
2. Скрипт применит настройки сверху файла и покажет отчёт

## Настройки

Настройки находятся в начале `очистка_комментариев_py.py`:

- `REMOVE_DOCSTRINGS`
- `REMOVE_QSS_CSS_COMMENTS`
- `REMOVE_EMPTY_COMMENT_LINES`
- `MAKE_BACKUP`
