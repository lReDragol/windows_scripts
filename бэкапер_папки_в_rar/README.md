# бэкапер_папки_в_rar

GUI-утилита на **PySide6** для бэкапа выбранной папки в архив **RAR**.

Что делает:

- собирает файлы по расширениям (настраивается)
- умеет исключать файлы/папки по паттернам
- создаёт (или находит) папку `!backup` в корне проекта
- создаёт `.rar` архив через внешнюю команду `rar`

## Требования

- Windows + установленный **WinRAR/RAR**
- команда `rar` должна быть доступна в `PATH` (или `rar.exe` лежит рядом со скриптом)
- Python + PySide6:

```bash
pip install PySide6
```

## Запуск

```bash
python бэкапер_папки_в_rar/бэкапер_папки_в_rar.py
```

## Примечания

- Если видишь ошибку «rar не найден», проверь установку WinRAR и переменную `PATH`.
