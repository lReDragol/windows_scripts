# переключатель_аккаунтов_codex

GUI-утилита на **PySide6** для управления профилями Codex:

- хранит несколько профилей `auth.json`
- позволяет быстро переключаться между ними
- делает бэкапы
- (в WSL) умеет синхронизировать `auth.json` между Windows и WSL

## Требования

```bash
pip install PySide6
```

## Запуск

```bash
python переключатель_аккаунтов_codex/переключатель_аккаунтов_codex.py
```

## Примечания

- Папка Codex по умолчанию: `~/.codex` (можно переопределить переменной `CODEX_HOME`).
- WSL синхронизация не трогает настройки, если задан `CODEX_HOME`.
