# windows_scripts

Набор утилит/скриптов (в основном под Windows).

## Список

- `плитки` — блокировка/разблокировка перемещения плиток меню «Пуск»: `плитки/README.md`
- `GrantFullAccess` — дать полный доступ к файлу/папке через PowerShell: `GrantFullAccess/README.md`
- `дерево_папок_gui` — GUI-утилита (PySide6) для построения дерева каталогов: `дерево_папок_gui/README.md`
- `бэкапер_папки_в_rar` — GUI-утилита (PySide6) для бэкапа папки в RAR (нужен `rar.exe`/WinRAR): `бэкапер_папки_в_rar/README.md`
- `очистка_комментариев_py` — GUI-утилита (Tkinter) для очистки комментариев в `.py`: `очистка_комментариев_py/README.md`
- `переключатель_аккаунтов_codex` — GUI-утилита (PySide6) для управления профилями Codex и синхронизации auth Windows ↔ WSL: `переключатель_аккаунтов_codex/README.md`
